# Greenpress Content service

A content service for greenpress platform 

## Main Features
- manage menus
- manage categories
- manage posts
- manage comments
- manage website configurations
- multi tenancy
- search posts
- post tags 
- auto-migrations for updates

## Dependencies
- Node.js
- npm OR yarn
- MongoDB
- @greenpress/auth service

## Install
```sh
$ npm install
```

## Launch
```sh
$ npm start
```
